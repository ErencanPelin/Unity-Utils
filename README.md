# Unity Utils
Some core unity scripts and modules that I can re-use across projects.
This project is used by my [UInit](https://github.com/ErencanPelin/UInit) tool so quickly import modules or utility scripts 
into any Unity Project, saving you from writing lots of boilerplate code or setting up the same systems for many games.

## Installation
There are a few ways to use this package, the recommended way is through the [UInit](https://github.com/ErencanPelin/UInit) tool.
This is a custom tool that I built to speed up my own development. Follow the instructions on UInit's README.

However, you can use the Unity Package manager:
1. In the Unity Package Manager Window, click 'Add Package' and 'Add Package by Git Url'. Enter the following: `https://github.com/ErencanPelin/Unity-Utils.git` and click `Add`.
2. This will add the package into your Packages folder.
